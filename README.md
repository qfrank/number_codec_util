#Number codec tool
  

###This tool class can encode number x(1<=x<=1000000000) to string which contains 6 chars using method **encode**.
###Encode result example: 
     1 => 9GFq89 
     2 => Kaoq8Z
     3 => XaARpR 
     4 => R53ulX 
     5 => 98qzy3 
     6 => dkhLY8 
     7 => bl84y8 
     8 => KMfvok 
     9 => dBIjfj 

###You can also decode this string to number again using method **decode**.
###Enjoy):
